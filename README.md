                                                                             ### راه‌اندازی پروژه

                                                                            1. ریپوی پروژه را کلون کنید:
##
git clone https://github.com/shirzadsara/currency.git
npm install
                                                                            یک فایل .env بسازید در ریشه پروژه و مقدار زیر را قرار دهید:
DATABASE_URL=your_database_url_here
                                                                            جای your_database_url_here آدرس دیتابیس خودتان را وارد کنید. فایل *.env* را روی GitHub قرار ندهید
                                                                            سرور را اجرا کنید:
npm run dev
